Widget Installation

Add this script to your website (just before the closing </body> tag):

<script src="http://localhost:3001/widget/vintrachat.js" data-widget-key="demo-widget-key-123" async></script>

Troubleshooting

  • 500 Internal Server Error: Make sure your Next.js server is running on port 3001
  • Widget not appearing: Check browser console (F12) for error messages
  • Invalid widget key: Make sure you're logged in to the dashboard and have a valid organization
  • CORS issues: The widget must be served from the same domain as your API
